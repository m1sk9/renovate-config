# @m1sk9/renovate-config

m1sk9 presets for [Renovate](https://renovatebot.com/).

## Usage

Add the following to your `renovate.json`:

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["github>m1sk9/renovate-config"]
}
```
